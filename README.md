# phucbui.me
## Simple Personal Landing Page Built With ExpressJS
* NodeJS
* ExpressJS
* SASS
* Jade

### Demo
https://phucbui.me

### Run
`npm start` for local serving

or

`npm install -g nodemon`

then

`npm run dev` for development serving

App will be running at `http://localhost:5000`
